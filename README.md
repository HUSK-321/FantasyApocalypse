# Project FA(FantasyApocalypse)

## 개요

스마일게이트 산하 퓨처랩에서 진행하는 UNSEEN 1기 활동의 개인 프로젝트 'Porject FA' 입니다. 언리얼 엔진 5를 활용해 루트 액션 멀티플레이 게임을 제작하는것이 목표입니다.
